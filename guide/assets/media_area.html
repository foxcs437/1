Tag Description

The area tag is used to define an area inside an image-map. An image-map is an image with clickable areas.

The area element is always nested inside a <map> tag.

Note: The usemap attribute in the <img> tag is associated with the <map> element's name attribute, and creates a relationship between the image and the map.

This tag has the form:

<area
  alt="text"
  coords="coordinates"
  href="URL"
  shape="{ circle | default |
    poly | rect }"
  target="{ _blank | _parent |
    _self | _top }"
/>

Required Attributes

Attribute Value Description

Examples

The following example shows the basic use of this tag:

<img src ="shapes.gif"
  width="150" height="80"
  alt="Shapes"
  usemap ="#shapemap" />
<map name="shapemap">
  <area shape="rect"
    coords="0,0,50,50"
    href="circle.htm"
    alt="Circle Shape" />
  <area shape="rect"
    coords="50,0,50,50"
    href="square.htm"
    alt="Square Shape" />
  <area shape="rect"
    coords="100,0,50,50"
    href="triangle.htm"
    alt="Triangle Shape" />
</map>
