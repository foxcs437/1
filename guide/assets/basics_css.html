Cascading Style Sheets (CSS)

Cascading Style Sheets (CSS) are a style sheet language used to describe the design of an HTML document. CSS can be used to control all visual aspects of the content. This includes:

  • Colors
  • Fonts
  • Layout
  • Much more

CSS is important to HTML because it enables the separation of content from design. CSS styles are usually separated into one of two places:

Internal CSS

External CSS

In general, using an external style sheet is preferred because it allows for site-wide consistency and reduces complexity / repetition.

A basic CSS rule consists of a selector and declarations and is written similar to the following:

p {
  color:red;
  font-family:Arial;
}

In this example, paragraph elements, p will have the two styles, color with a value of 'red' and font-family with a value of 'Arial' applied to them.
