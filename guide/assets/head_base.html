Tag Description

The base element is used to set a default address or a default target for all links on a page.

This tag has the form:

<base
  href="URL"
  target="{ _blank | framename |
     _parent | _self | _top } "/>

Note: The <base> tag must be placed within the <head> element.

Attributes

Attribute Value Description
href URL

Sets the base URL for all relative URLs on a page

Note: The base URL must be absolute
target _blank
_parent
_self
_top
framename		 Specifies where to open all the links on a page

Standard Attributes

None supported.

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<base href=
  "http://example.com" />
