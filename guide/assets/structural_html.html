Tag Description

The html tag is used to indicate to the browser that this document is an HTML document.

The <html> element is the outermost element in HTML and XHTML documents and is also known as the document's root element.

This tag has the form:

<html xmlns=
  "http://www.w3.org/1999/xhtml">
</html>

Note: In XHTML documents, the required xmlns attribute must be included in the head element and assigned a value of http://www.w3.org/1999/xhtml.

Required Attributes

Attribute Value Description
xmlns

http://www.w3.org/
1999/xhtml

 Specifies the namespace to use (only for XHTML documents)

Standard Attributes

Attribute Value Description
dir rtl
ltr		 Sets the text direction for the content of the element
lang lang_code Sets the language code for the content of the element
xml:lang lang_code Sets the language code for the content of the element in XHTML documents

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<!DOCTYPE ...
<html xmlns=
  "http://www.w3.org/1999/xhtml">
<head>
  <meta http-equiv="Content-Type"
    content="text/html;
    charset=utf-8" />
  <title>Web Page Title</title>
</head>
<body>
  <p>Document content...</p>
</body>
</html>
