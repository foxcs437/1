Tag Description

The style tag is used to define style information for an HTML document.

This tag has the form:

<styleÂ media="media types"
  type="MIME_type">
</style>

Note: The <style> tag must be placed within the <head> element.

Required Attributes

Attribute Value Description
type text/css Specifies the MIME type of the style sheet

Attribute Value Description

Standard Attributes

Attribute Value Description
dir rtl
ltr		 Sets the text direction for the content of the element
lang lang_code Sets the language code for the content of the element
title text Sets extra information about the element
xml:lang lang_code Sets the language code for the content of the element in XHTML documents

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<head>
<style type="text/css">
  h1 {color:red}
  p {color:blue}
</style>
</head>
