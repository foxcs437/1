Tag Description

The colgroup tag is used to group columns in a table for formatting.

This tag has the form:

<colgroup
  align="{ center | char |
    justify | left | right }"
  span="number"
  valign="{ baseline | bottom |
    middle | top }"
  width="{ number |
    percentage }">
</colgroup>

Note: This tag must appear within <table></table> tags.

Examples

The following example shows the basic use of this tag:

<table width="100%" border="1">
  <colgroup span="2"
    align="left">
  </colgroup>
  <colgroup align="right">
</colgroup>
  <tr>
    <th>H1</th>
    <th>H2</th>
    <th>H3</th>
  </tr>
  <tr>
    <td>Item 1</td>
    <td>Item 2</td>
    <td>Item 3</td>
  </tr>
</table>

This produces the following result:

H1 H2 H3
Item 1 Item 2 Item 3
