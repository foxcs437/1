Tag Description

The body element wraps around all of the HTML elements that will be rendered to the screen or other media.

This tag has the form:

<body>
  html elements
</body>

Examples

The following example shows the basic use of this tag:

<body>
  <h2>First Heading</h2>
  <p>Paragraph 1</p>
</body>
