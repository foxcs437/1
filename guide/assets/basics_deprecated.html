Deprecated Elements

The following list of elements are deprecated, meaning that they have been removed in later revisions of standards documents. Because of this, their use should be avoided due to possible future lack of support.

Element Description
applet Defines an embedded applet
basefont Defines a default font, color, or size for the text in a page
center Defines centered text
dir Defines a directory list
font Defines font, color, and size for text
isindex Defines a searchable index related to a document
menu Defines a menu list
s Defines strikethrough text
strike Defines strikethrough text
u Defines underlined text
xmp Defines preformatted text
