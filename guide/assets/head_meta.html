Tag Description

The meta tag is used to provide metadata about the HTML document. Metadata is not rendered on the page, but is browser parsable.

Metadata elements are typically used to specify the document description, keywords, and author information.

This tag has the form:

<meta content="text" {
  http-equiv=
    "http response header" |
    name="text" }
/>

Note: The <meta> tag must be placed within the <head> element.

Required Attributes

Attribute Value Description
content text Specifies the content of the meta information

Optional Attributes

Attribute Value Description
http-equiv content-type
content-style-type
expires
refresh

name author

generator

others		 Provides a name for the information in the content attribute

Standard Attributes

Attribute Value Description
dir rtl
ltr		 Sets the text direction for the content of the element
lang lang_code Sets the language code for the content of the element
xml:lang lang_code Sets the language code for the content of the element in XHTML documents

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<head>
<meta name="description"
  content="Example web site" />
<meta name="keywords"
  content="Example, Test" />
<meta name="author"
  content="John Q. Public" />
<meta http-equiv="Content-Type"
  content="text/html;
  charset=ISO-8859-1" />
</head>
