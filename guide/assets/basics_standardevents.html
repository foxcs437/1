Standard Event Attributes

HTML 4 added the ability to let events trigger actions in a browser. These actions are carried out by a client-side scripting language such as JavaScript.

The tables below list the standard event attributes that can be inserted into HTML / XHTML elements to define event actions.

<body> and <frameset> Events

The two attributes below can only be used in <body> or <frameset>:

Attribute Value Description
onload script Executes script when a document loads
onunload script Executes script when a document unloads

Form Events

The attributes below can be used in form elements:

Attribute Value Description
onblur script Executes script when an element loses focus
onchange script Executes script when an element's value changes
onfocus script Executes script when an element gets focus
onreset script Executes script when a form is reset
onselect script Executes script when an element is selected
onsubmit script Executes script when a form is submitted

Image Events

The attribute below can be used with the <img> element:

Attribute Value Description
onabort script Executes script when loading of an image is interrupted

Keyboard Events

Valid in all elements except: <base>, <bdo>, <br>, <frame>, <frameset>, <head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, <title>.

Attribute Value Description
onkeydown script Executes script when a key is pressed
onkeypress script Executes script when a key is pressed and released
onkeyup script Executes script when a key is released

Mouse Events

Valid in all elements except: <base>, <bdo>, <br>, <frame>, <frameset>, <head>, <html>, <iframe>, <meta>, <param>, <script>, <style>, <title>.

Attribute Value Description
onclick script Executes script on a mouse click
ondblclick script Executes script on a mouse double-click
onmousedown script Executes script when a mouse button is pressed
onmousemove script Executes script when the mouse cursor moves
onmouseout script Executes script when the mouse cursor moves out of an element
onmouseover script Executes script when the mouse cursor moves over an element
onmouseup script Executes script when a mouse button is released
