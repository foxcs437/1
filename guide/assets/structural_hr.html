Tag Description

The hr tag is used to create a horizontal line in an HTML document. The horizonal line visually separates content.

This tag has the form:

<hr />

Optional Attributes

Attribute Value Description

right

noshade noshade (deprecated) Specifies that the element should be drawn using a solid color, instead of a shaded color
size pixels (deprecated) Sets the height of the hr element
width pixels
%		 (deprecated) Sets the width of the hr element

Examples

The following example shows the basic use of this tag:

<p>Paragraph 1</p>
<hr />
<p>Paragraph 2</p>

This produces the following:

Paragraph 1

Paragraph 2
