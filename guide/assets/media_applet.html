Tag Description

The applet tag is used to define an embedded applet.

This tag has the form:

<applet
  archive="URL"
  code="URL"
  codebase="URL"
  height="{ number |
    percentage }"
  width="{ number |
    percentage }"
>  
</applet>

Tip: To include applets, use the <object> tag.

Required Attributes

Attribute Value Description
code URL Specifies the file name of a Java applet
object name Specifies a reference to a serialized representation of an applet

Optional Attributes

Attribute Value Description
align left
right
top
bottom
middle
baseline		 Specifies the alignment of an applet according to surrounding elements
alt text Specifies an alternate text for an applet
archive URL Specifies the language of the text in the linked document
codebase URL Specifies a relative base URL for applets specified in the code attribute
height pixels Specifies the height of an applet
hspace pixels Defines the horizontal spacing around an applet
name pixels Defines the name for an applet (to use in scripts)
vspace pixels Defines the vertical spacing around an applet
width pixels Specifies the width of an applet

Standard Attributes

Attribute Value Description
class classname Specifies a classname for an element
id id Specifies a unique id for an element
style id Specifies an inline style for an element
title style_defenition Specifies extra information about an element

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<applet code="example.class"
  width="200" height="200">
  Example Java applet
</applet>
