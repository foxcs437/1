Tag Description

The img tag is used to link an image to an HTML page. The tag creates a holding space for the referenced image.

This tag has the form:

<img
  alt="text"
  src="URL"
  height="{ number |
    percentage }"
  width= "{ number |
    percentage }"
/>

Required Attributes

Attribute Value Description
alt text Specifies alternate text for the image
src URL Specifies the URL of the image

Optional Attributes

Attribute Value Description
align top
bottom
middle
left
right		 (deprecated) Specifies the alignment of an image according to surrounding elements
border pixels (deprecated) Specifies the width of the border around an image
height pixels
%		 Specifies the height of an image
hspace pixels (deprecated) Specifies the whitespace on left and right side of an image
ismap ismap Specifies an image as a server-side image-map. Refer to <usemap> instead
longdesc URL Specifies the URL to a document that contains a long description of the image
usemap #map Specifies an image as a client-side image-map
vspace pixels (deprecated) Specifies the whitespace on top and bottom of an image
width pixels % Specifies the width of the image

Standard Attributes

onmouseover script Executes script when the mouse cursor moves over the element
