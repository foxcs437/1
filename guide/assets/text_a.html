Tag Description

The a tag is used to define an anchor (usually referred to as a link or a hyperlink).

An anchor can be used to 1) create a link to another document, by using the href attribute, or 2) create a bookmark inside a document, by using the name attribute.

This tag has the form:

<aÂ href="URL">
</a>

By default, links appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red
