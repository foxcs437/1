Tag Description

The head tag is used to wrap around all of the head elements. Head elements set the document title (title), instruct the browser where to find style sheets (style), define link relationships (link), set meta information (meta), or include scripts (script).

This tag has the form:

<head>
  head elements
</head>

Optional Attributes

Attribute Value Description
profile URL Specifies a URL to a document that contains a set of rules. The rules can be read by browsers to clearly understand the information in the <meta> tag's content attribute

Standard Attributes

Attribute Value Description
dir rtl
ltr		 Sets the text direction for the content of the element
lang lang_code Sets the language code for the content of the element
xml:lang lang_code Sets the language code for the content of the element in XHTML documents

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<html>
<head>
  <meta
    http-equiv="Content-Type"
    content="text/html;
    charset=utf-8" />
  <title>Web Page Title</title>
</head>
<body>
  <p>Document content...</p>
</body>
</html>
