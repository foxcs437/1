URL Encoding

Browsers request web pages from web servers by using a URL (Uniform Resource Locator). URLs look something like:

http://www.samplewebsite.com

URL Encoding is important to the HTML coder since all URLs in an HTML document used to reference or import an object should be encoded. This includes the use of tags: <a>, <area>, <base>, <body>, <form>, <frame>, <iframe>, <img>, <link>, <script>, and others.

To be used over the internet, URLs can only contain certain characters from the ASCII character set. Since URLs often contain characters outside the ASCII set, the URL has to be converted.

Invalid URLs are converted into valid ASCII format by replacing the invalid characters with "%" followed by two hexadecimal digits corresponding to the character values in the ISO-8859-1 character set.

Note: URLs cannot contain spaces. Spaces are normally replaced with %20.
