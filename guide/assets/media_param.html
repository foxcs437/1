Tag Description

The param tag is used to define variables or parameters for an <applet> or <object> element.

This tag has the form:

<param name="text"
  type="MIME_type"
  value="value"
  valuetype="{ data | object |
    ref }">

Required Attributes

Attribute Value Description
name name Sets the parameter name

Optional Attributes

Attribute Value Description
type MIME_type Specifies the MIME type for a parameter
value value Specifies the value of a parameter
valuetype data
ref
object		 Specifies the type of the value

Standard Attributes

Attribute Value Description
id id Specifies a unique id for an element

Event Attributes

None supported.

Examples

The following example shows the basic use of this tag:

<object
  codebase="http://www.xyzzy.com/
    applet.class"
  classid="java:program.start"
  codetype="application/java-vm"
  height="50"
  width="50">
  <param name="options"
    value="xyzzy" />
  Browser does not support Java.
</object>
